# CourtagePro — Base Premium (démo)

## Lancer
```bash
npm install
npm run dev
```

Ouvrir:
- http://localhost:3000/ (Accueil)
- http://localhost:3000/devis (Formulaire -> enregistre en SQLite)
- http://localhost:3000/admin (Dashboard admin -> lit les demandes réelles)
- http://localhost:3000/client (UI démo)
- http://localhost:3000/artisan (UI démo)

## Base de données
- Fichier: `database.sqlite` (créé automatiquement au 1er lancement)

## Prochaine étape (V2)
- Auth (client / artisan / admin)
- Profils artisans (zones + métiers)
- Matching automatique + distribution des demandes
- Messagerie + upload photos
- Paiement (abonnement / lead / commission)
